Strength Reduction

Optimization: Replace expensive operations with equivalent but cheaper ones.

    result += 2 * i;
}



for (int i = 0; i < n; i++) {
    result += i << 1;
}

Loops

Loop Unrolling:

Optimization: Reduce loop overhead by duplicating loop bodies.


for (int i = 0; i < 4; i++) {
    arr[i] = i * 2;
}



arr[0] = 0 * 2;
arr[1] = 1 * 2;
arr[2] = 2 * 2;
arr[3] = 3 * 2;

Loop Fusion and Fission

Loop fusion combines multiple loops into one to reduce overhead. Loop fission splits a loop into multiple loops to improve cache usage or parallelism.


for (int i = 0; i < n; i++) {
    // Loop 1 body
}
for (int i = 0; i < n; i++) {
    // Loop 2 body
}



for (int i = 0; i < n; i++) {
    // Combined Loop body
}

Loop-Invariant Code Motion (LICM)

Optimization: Move calculations that do not depend on loop variables out of the loop to reduce redundancy.


for (int i = 0; i < N; i++) {
    result[i] = x + y;
}



int temp = x + y;
for (int i = 0; i < N; i++) {
    result[i] = temp;
}
