Turtle commands

Command Description
Turtle.carft(number quantity) Craft items using ingredients anywhere in the turtle's inventory and place results in the active slot. If a quantity is specified, it will craft only up to that many items, otherwise, it will craft as many of the items as possible.
turtle.forward() | .back() | .up() | .down() Movement
turtle.turnLeft() | .turnRight() turning
turtle.select(number slot) Make the turtle select slot slotNum (1 is top left, 16 (9 in 1.33 and earlier) is bottom right)
turtle.getSelectedSlot() Indicates the currently selected inventory slot
turtle.getItemCount([number slotNum]) Counts how many items are in the currently selected slot or, if specified, slotNum slot
turtle.getItemSpace([number slotNum]) Counts how many remaining items you need to fill the stack in the currently selected slot or, if specified, slotNum slot
turtle.getItemDetail([number slotNum]) Returns the ID string, count and damage values of currently selected slot or, if specified, slotNum slot
turtle.equipLeft() | .equipRight() Attempts to equip an item in the current slot to the turtle's side, switching the previously equipped item back into the inventory
turtle.attack([string toolSide]) Attacks
turtle.dig([string toolSide]) Breaks the block. With hoe: tills the dirt.
turtle.place([string signText]) Places a block of the selected slot in front. Engrave signText on signs if provided. Collects water or lava if the currently selected slot is an empty bucket.
turtle.detect() Detects if there is a block in front. Does not detect mobs.
turtle.inspect() Returns the ID string and metadata of the block in front of the Turtle
turtle.compare() Detects if the block in front is the same as the one in the currently selected slot
turtle.compareTo(number slot) Compare the current selected slot and the given slot to see if the items are the same. Returns true if they are the same, false if not.
turtle.drop([number count]) Drops all items in the selected slot, or specified, drops count items.
turtle.suck([number amount]) Picks up an item stack of any number, from the ground or an inventory in front of the turtle, then places it in the selected slot. If the turtle can't pick up the item, the function returns false. amount parameter requires ComputerCraft 1.6 or later.
turtle.refuel([number quantity]) Refuels
turtle.getFuelLevel() Returns the current fuel level of the turtle, this is the number of blocks the turtle can move.
turtle.transferTo(number slot [, number quantity]) Transfers quantity items from the selected slot to slot. If quantity isn't specified, will attempt to transfer everything in the selected slot to slot.

                 -- Monitores Buttons and graphics
w, h = term.getSize()
monitor = peripheral.wrap("top")

monitor.setTextScale(0.5)
monitor.clear()

local buttons = {
    { text = "Button 1", x = 1, y = 1, w = 10, h = 3 },
    { text = "Button 2", x = 1, y = 5, w = 10, h = 3 },
    { text = "Button 3", x = 1, y = 9, w = 10, h = 3 },
}

local function drawButton(button)
    monitor.setCursorPos(button.x, button.y)
    monitor.write(string.rep(" ", button.w))
    monitor.setCursorPos(button.x, button.y)
    monitor.write(button.text)
end

for _, button in ipairs(buttons) do
    drawButton(button)
end

while true do
    local event, side, x, y = os.pullEvent("monitor_touch")
    for _, button in ipairs(buttons) do
        if x >= button.x and x <= button.x + button.w - 1 and
              y >= button.y and y <= button.y + button.h - 1 then
            monitor.clear()
            monitor.setCursorPos(1, 1)
            monitor.write("You clicked "..button.text)
            sleep(1)
            monitor.clear()
            for _, button in ipairs(buttons) do
                drawButton(button)
            end
            break
        end
    end
end
