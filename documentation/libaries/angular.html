Basics

Installation

            
/** Node.js must be Installed */
npm install -g @angular/cli

Init

            
ng new appName  /** Init project */
ng generate component componentName

Run

            
ng serve --open

// or
ng add @angular/pwa
ng build
ng serve --open
// oben ist das install icon

Project must be in C directory

Code

    
<button (click)="func()"> </button> // on click execute
    

    
<p *ngIf="count <= 1> Text </p>     // only shows if condition is true
    

    
<p> {{ varName }} </p> // prints variable
