Booleans represent one of two values: True or False.

Implementation

                
import re
text = "..."
pattern = re.compile(r'abc')

matches = pattern.finditer(text)

for match in matches:
    print(match)
Symbol Description
. Everything but a new line
\d Digits 0 - 9
\D Not a digit
\w (a-z, A-Z,0-9,_)
\W Not a "normal" character
\s Whitespace (space, tab, new line)
\S Not a Whitespace
\b Word Boundary (space bevore)
^ Start of string
$ End of string
Symbols Description
[ ] Matches character
[^ ] Not in brackets
| Either or
{3,4} Range
{3} Exact
* 0 or more
+ 1 or more
? 0 or 1
